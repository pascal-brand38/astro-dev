## Install

```bash
npm install astro-lightgallery
```

Read the full doc on
[Github](https://github.com/pascal-brand38/astro-lightgallery/blob/main/README.md)

LightGallery documentation can be found on [lightgalleryjs.com](https://www.lightgalleryjs.com/)

  import { LightGallery } from 'astro-lightgallery'
  ---

  <LightGallery
    class="..."
    options={{
      thumbnail: true,
      autoplay: true,
      slideShowInterval: 500,
      }}
    addPlugins={[ 'zoom' ]}
    >
    {
      imgs.map(img => (
        <a class="..." href={img.srcFull}>
            <img src={img.srcFull} />
        </a>
      ))
    }
  </LightGallery>
