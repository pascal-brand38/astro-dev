```bash
npm install astro-swiper
```

Read the full doc on
[Github](https://github.com/pascal-brand38/astro-swiper/blob/main/README.md)

Swiper options documentation can be found on [swiperjs.com](https://swiperjs.com/swiper-api#parameters)
